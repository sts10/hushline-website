Get Hush Line

Your Private Tip Line

Receive encrypted messages that only you can read.

Hush Line is a free and open-source app to host a private tip line enabling anyone to send you encrypted messages without requiring technical expertise, downloading any new software, generating keys, or creating an account.

Who Is Hush Line For?

Journalists / Newsrooms

Provide the public with a safe way to leave you a confidential tip.

Employers / Boards

Build trust by providing an anonymous way for colleagues to leave suggestions.

Educators / School Staff

Host a safe way for students to share information with someone they trust.

Receive Encrypted Messages

Use any public PGP key to encrypt messages so only the key's owner can read them.

No Account Needed

Hush Line deploys to a publicly available website and doesn't require a user to create an account to leave a message.

Privacy by Default

Hush Line uses privacy-preserving logging, which removes visitors' IP addresses.

Tor Ready

Automatically deploy a Tor onion service so users can privately use your app with confidence.

Security by Default

Hush Line configures your server's content security headers to minimize your app's attack surface.
